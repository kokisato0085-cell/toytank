# ToyTank

### ▶ ブラウザでそのままプレイ → **https://kokisato0085-cell.github.io/toytank/**
（スマホ／PC対応・インストール不要。ステージ作成は [ToyTank Maker](https://kokisato0085-cell.github.io/toytank/editor.html)）

スマホのブラウザで遊べる **2D見下ろし戦車バトル**。
弾は壁で跳ね返り（跳弾）、地雷を仕掛け、敵戦車を全滅させる。
ソロ、またはオンライン2人協力（デュオ）で同じステージに挑める。

> 「はじめてのWii / Wii Play」収録の「戦車！」へのオマージュ作品です。
> 名称・画像・音はすべて独自に用意します（既存IPの素材は使用しません）。
> 画像素材（タイル・戦車など）は **Google Gemini で生成**したものを使用しています。

## 特徴
- **タッチ操作（PC切替対応）**：左スティックで移動、右エイムパッドを引いて離した瞬間に発射。PCではマウスカーソル照準にも切替可能。
- **跳弾の物理**：弾は壁で反射。敵AIは遮蔽物の向こうのプレイヤーを、外壁・内壁を使った反射（バンクショット）で狙ってくる。
- **色分けされた敵タイプ**：色ごとに速度・思考（待機/接近/カイト/追跡）・弾速・反射回数・地雷の有無などが異なる10種以上。BFS経路探索で壁を避けて動く。
- **地雷・破壊可能な壁**：地雷は時限で爆発し戦車も壁も巻き込む。爆風に触れた弾も誘爆して消える。もろい木の壁は爆発で壊れる。設置直後でも爆風内なら設置者本人を巻き込む（自分の地雷で弾を受け止める“盾”はできない）。
- **ボス戦**：体力バー付きの大型ボス。被弾してやり直し（残機-1）になっても、削ったボスの体力は引き継がれる。
- **データ駆動ステージ ＋ 自作エディタ「ToyTank Maker」**：マップと敵配置を JSON で持ち、エディタでマウス作成できる。同梱の20面キャンペーンを順番に遊べる。
- **演出と効果音**：戦車スプライト・タイル画像、撃破演出、キャタピラ跡、効果音（発射/跳弾/爆発/走行音ほか）。
- **フレンドリーファイアあり**：味方の弾も味方に当たる（協力プレイ向けの設計）。敵AIは射線上に味方の敵戦車がいると、同士討ちを避けて撃たない。

## 開発ノート

実装を進める中で「工夫したところ」と「苦戦したところ」を、テーマごとに記録していく。

### 工夫したところ

- **データ駆動の敵タイプ**：色ごとに速度・思考（待機/接近/カイト/追跡/バランス）・発射間隔・弾数・弾速・反射回数・地雷数・HP・透明化などをテーブルで定義。新しい敵を増やすときはコードではなく値を足すだけ。
- **バンクショットの軌道シミュレーション**：敵は遮蔽越しでも、実弾と同じ反射物理をなぞって命中する発射角を探す。外壁・内壁すべての反射を使い、弾の反射回数（例：黄緑＝2回）に応じた跳ね返しも狙う。
- **BFS経路探索で壁を避ける動き**：敵は目標セルまでの最短経路を辿り、壁際でも擦りながら回り込む。隅にハマったら一定時間で無目的移動へ切り替えて脱出。
- **弾の相殺制御（同士討ち防止）**：同じ一斉射の弾を「グループ」で識別し、同グループは相殺しないが別グループ（自分の弾どうしを含む）は相殺。連射やボスの斉射が自滅しないようにした。
- **タイル/スプライト描画**：床は1枚を全面に敷いて継ぎ目を消し、壁・穴は画像の不透明部分を自動で切り出してマスにフィット。戦車は白シルエットに色を乗算してタイプ色に塗り分け（陰影は保持）。
- **音まわりの作り込み**：効果音は読み込み時に先頭無音を自動検出してスキップ、連続音はスロットルで間引き、初回タップで自動再生制約を解除。走行音はフェードでON/OFFするループ。BGMは被弾で停止し復活で頭から再開、クリアで終了、ステージ移行では継続。
- **同士討ちを避ける敵AI**：敵は発射前に弾道（直射・跳弾の反射経路を含む）をシミュレートし、プレイヤーへ届く手前に味方の敵戦車がいると発射を見送る。乱戦でも仲間を撃たない。
- **手触りの演出**：撃破時の白いバッテン印、キャタピラ跡、戦車の大破爆発、ボスの体力バー（残機-1後も削った体力を継続）、ステージ背景を残したままのクリアポップアップ（総撃破数表示）。

### 苦戦したところ

- **跳弾の読み合い**：1回反射前提で撃つと、2回反射する弾は2発目で曲がって当たらない。鏡像法では内壁を絡めた反射を表現できず、最終的に軌道シミュレーション方式へ作り替えた。
- **自滅の制御と地雷ガードの悪用対策**：ボスを2倍サイズにしたら砲口が機体内に入って自分の弾で自爆、紫が自分の地雷で自爆…と多発。発射位置を機体の外側から出す、同グループ弾は相殺しない、と一つずつ条件を詰めた。地雷には当初「設置直後は設置者が無傷」の起動猶予を入れていたが、敵弾を地雷で受け止めて無傷になる“盾”として悪用できたため撤廃し、爆風内なら設置者本人も巻き込むようにした。
- **敵の動きの調整**：壁に引っかかる・隅でハマる・追跡が強すぎる、など何度も調整。経路探索＋スタック脱出＋性格ごとの行動切替（接近/カイト/追跡）でバランスを取った。
- **敵の性能バランス**：色ごとの発射レート・弾速・連射の比率・地雷数を、テストプレイのたびに細かく上げ下げ。「連射＝同じ発射口から最速で連続発射」「近いほど連射が伸びる」など挙動の意味づけも調整した。
- **画像素材の扱い**：生成画像の透明背景・緑フチ・サイズ違い・タイルの継ぎ目や四隅の黒抜けに対応。表示サイズや切り出し範囲を実物で合わせ込んだ。
- **音の鳴り出しと自動再生制約**：素材の頭に無音があって遅れて聞こえる、スマホは最初のタップまで鳴らない、といった制約に、先頭無音スキップとユーザー操作での解除で対応。
- **操作感**：戦車（キャタピラ）らしく進行方向の逆へは即バック、発射の瞬間だけ一瞬停止、PC（マウス照準）とスマホ（スティック）の操作切替など、感触に関わる部分を繰り返し微調整。
- **暴発・取りこぼし**：開始前カウントダウンやミス待機中のクリックが再開と同時に発射される問題を、非プレイ中の入力破棄で解消。
