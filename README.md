# ToyTank

スマホのブラウザ（PWA）で遊べる **2D見下ろし戦車バトル**。
弾は壁で跳ね返り（跳弾）、地雷を仕掛け、敵戦車を全滅させる。
ソロ、またはオンライン2人協力（デュオ）で同じステージに挑める。

> 「はじめてのWii / Wii Play」収録の「戦車！」へのオマージュ作品です。
> 名称・画像・音はすべて独自に制作します（既存IPの素材は使用しません）。

## 特徴
- **ブロスタ風のタッチ操作**：左スティックで移動、右エイムパッドを引いて離した瞬間に発射。
- **跳弾の物理**：弾は壁で反射。敵AIは遮蔽物の向こうのプレイヤーを跳弾で狙ってくる。
- **データ駆動ステージ ＋ 自作エディタ「ToyTank Maker」**：マップと敵配置を JSON で持ち、エディタでマウス作成できる。
- **フレンドリーファイアあり**：味方の弾も味方に当たる。協力ならではの緊張感。

## 技術スタック
- クライアント：TypeScript + HTML5 Canvas + Vite（PWA）
- リアルタイムサーバー（段階2のオンライン協力以降）：Go
- テスト：Vitest（クライアント）、`go test`（サーバー）
