# 効果音（SE）置き場

ここに `<名前>.mp3` を置くと、ゲームが自動で読み込んで鳴らす。無ければ無音でフォールバックする。
Vite が `public/` をルート配信するので、ゲームからは `/audio/<名前>.mp3` で参照する。

## 必要なファイル（短いSE推奨・mp3）

| ファイル | 使う場面 | 目安 |
|---|---|---|
| `shot.mp3` | 弾を撃った瞬間（自機・敵とも） | ごく短い発射音（0.1〜0.3秒） |
| `bounce.mp3` | 弾が壁で跳ね返った瞬間 | 短い金属音（0.1秒前後） |
| `mine.mp3` | 地雷を設置した瞬間 | 短いカチッ音 |
| `explosion.mp3` | 地雷の爆発・戦車の大破（共用） | 爆発音（0.3〜0.6秒） |
| `miss.mp3` | 自機がやられて残機-1 | ミス/被弾音 |
| `clear.mp3` | ステージクリア | 短いクリアジングル |
| `gameover.mp3` | 全滅（ゲームオーバー） | ゲームオーバー音 |

## 仕様メモ
- 形式は **mp3**（拡張子固定）。ステレオ/モノどちらでも可。
- 連続して鳴るSE（`shot`/`bounce`/`explosion`）はゲーム側で短時間のスロットルを掛けて間引く。
- スマホは最初の画面タップまで音が鳴らない（ブラウザの自動再生制約）。タップ後に有効化される。
- 画面右下の「🔊 音 ON / 🔇 音 OFF」ボタンでミュート切替（設定は記憶）。
