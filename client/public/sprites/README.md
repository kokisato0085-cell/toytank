# スプライト置き場（グラフィック）

> 画像素材は **Google Gemini で生成**して配置しています。

ここに画像（PNG・背景透過）を置くと、ゲームが自動で使う。無ければ従来の図形描画にフォールバックする。
Vite が `public/` をルート配信するので、ゲームからは `/sprites/<名前>.png` で参照する。

## 必要なファイルと仕様

### タイル（マップ）— 各 64×64px 推奨（セルサイズ）
- `floor.png` … 床
- `steel.png` … 壊せない壁（鋼）
- `brick.png` … 壊せる壁
- `hole.png` … 穴（グリッド内に収まる円＝周囲は床。背景透過なら床の上に重ねて描く）

### 戦車 — 車体と砲塔を分けた2枚（1枚を色で着色）
どちらも **真上(北)向き・白〜薄グレーの単色シルエット・背景透過・中央配置**。サイズ目安 48×48px（車体直径=48）。ゲーム側でタイプ色に着色し、ボスは自動2倍。
- `tank_body.png` … **車体（キャタピラ/シャシー）**。**移動方向**に合わせて回転する。
- `tank_turret.png` … **砲塔（砲身）**。**照準方向**（射線が通る時は自機/プレイヤーは右パッド方向）に合わせて回転する。車体の上に重ねて描画。

> 着色は「白〜グレーに色を乗算」する方式。陰影を付けたいなら薄いグレーで描けば立体感が残る。真っ白だとベタ塗りの色になる。

### 任意（無ければ今の描画のまま）
- `bullet.png` / `mine.png` … 弾・地雷（無くてOK）

## 注意
- 透過PNG推奨。タイルはセル(64px)にぴったり敷き詰める前提。
- 戦車は「上向き」で描けば、ゲーム側で砲塔の向きに合わせて回転する。
